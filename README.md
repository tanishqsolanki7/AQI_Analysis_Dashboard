# AQI_Analysis_Dashboard
Project Overview

This project presents a comprehensive Air Quality Index (AQI) Analysis Dashboard built in Power BI using merged datasets from Data.gov.in and Kaggle.
The dashboard provides a detailed overview of India’s air quality by analyzing pollutant concentrations, state-wise AQI distribution, monthly pollutant trends, and AQI category percentages.

The objective of this project is to transform raw environmental data into meaningful insights through data cleaning, preprocessing, DAX calculations, and interactive visual storytelling.

Key Features
State-wise AQI Map Visualization for geographical AQI distribution across India
Top 10 States with Highest AQI using comparative pollutant breakdown
Air Quality Volume Distribution by AQI categories
Monthly AQI Trends for pollutants (PM10, CO, NO2, Ozone, SO2, NH3)
Pollutant Spread Analysis for contribution comparison
KPI Cards displaying percentage share of air quality status

Data Sources
Data.gov.in – Official Indian environmental datasets
Kaggle – Supplementary AQI and pollutant datasets
Data Cleaning & Transformation

The following preprocessing steps were performed before dashboard creation:

Merged multiple datasets from different sources
Removed null/missing values
Standardized column names and data types
Created custom AQI Quality Category column
Cleaned inconsistent pollutant labels
